This repository contains the data and code used to generate the results presented in the research paper "Predicting temperatures in Brazilian states capitals via Machine Learning". 
Available on: https://arxiv.org/abs/2505.11511

The folder Data_codes contains two folders: 
Data: Store the Normal, Reanalysis, and anomalous data. Each folder has the respective state data with some codes as examples.
- ML: stores two files that run the Random Forest algorithm once and using a loop.
