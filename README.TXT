This repository contains the data and code used to generate the results presented in the research paper "Predicting temperatures in Brazilian states capitals via Machine Learning". 
Available on: https://arxiv.org/abs/2505.11511

The folder Data_codes contain two folders: 
- Data: storage the Normal, Reanalisis and anomalous data. Each folder has the respective state data with some codes as examples.
- ML: storage two files which runs the Random Forest algorithm once and using a loop.
